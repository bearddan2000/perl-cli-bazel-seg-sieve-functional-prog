# perl-cli-bazel-seg-sieve-functional-prog

## Description
Given a range find all primes.
Follows functional programming
practices.

## Tech stack
- perl 5
- bazel

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
